# Responsive Website - Advanced CSS Styling

## Project Description
This project demonstrates **Advanced CSS Styling and Responsive Design** using **HTML, CSS, and Bootstrap**.  
The webpage contains multiple sections and uses animations, transitions, and responsive layouts to work on different screen sizes.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5

## Features
- Responsive layout using Bootstrap Grid
- Multiple sections (Navbar, Hero, About, Services, Contact)
- CSS transitions and hover animations
- Gradient hero section
- Mobile responsive design using media queries
